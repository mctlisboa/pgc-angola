- Accounts marked with `___` (e.g., `Banco ___`) are placeholders to be customized per entity
- Sub-accounts can be added as needed following the hierarchical numbering system

## Hierarchical Numbering

The PGC uses a hierarchical decimal numbering system:

- **Class level**: Single digit (1-9)
- **Account level**: Two digits (11, 21, 31, etc.)
- **Sub-account levels**: Additional digits after decimal points (11.1, 11.1.1, 11.1.4.1, etc.)

Example hierarchy:
```
1                    Class: Fixed assets and investments
├─ 11                Account: Tangible fixed assets
│  ├─ 11.1           Sub-account: Land and natural resources
│  │  ├─ 11.1.1      Sub-sub-account: Raw land
│  │  └─ 11.1.2      Sub-sub-account: Improved land
│  └─ 11.2           Sub-account: Buildings and structures
```
